# B-0301 — Repo Scaffold + 기본 문서 배치

## Goal
Reliability Lab 레포의 기본 문서/폴더 구조를 생성하고, 모든 실험 티켓이 기대하는 경로를 고정한다.

## Scope
- 폴더 구조 생성
- README.md / AGENTS.md / Plans.md / ARCHITECTURE.md / Runbook.md 생성(초안은 본 티켓 첨부 내용을 사용)
- `tasks/backlog|doing|done` 구조 생성
- `experiments/` + `scripts/` 디렉토리 생성

## Deliverables
- 루트 문서 5개 커밋
- 빈 실험 문서 템플릿 `experiments/_template.md`
- 빈 스크립트 템플릿 `scripts/exp`(B-0320에서 구현 예정)

## Acceptance Criteria
- `ls` 기준 구조가 문서와 일치
- 문서 링크들이 404 없이 서로 참조됨
