# B-0362 — E-044 ElastiCache Online Resharding/Shard Rebalancing under Load

## Goal
Cluster mode enabled 환경에서 온라인 리샤딩/리밸런싱 중에도 SLO를 만족하도록 운영 절차를 확정한다.

## Scope
- 실험 문서: `experiments/E-044-elasticache-online-resharding-under-load.md`
- 부하 하에서 scale-out + rebalance 수행
- 관측 항목 수집:
  - p95/p99 latency, timeout/retry, throughput/error
  - MOVED/ASK 비율(클라이언트 로그)
  - CloudWatch: EngineCPUUtilization, CurrConnections, FreeableMemory, Evictions, NetworkBytesIn/Out, (가능하면)ReplicationLag
- 완화책 비교:
  - timeout/retry/backoff+jitter
  - topology refresh 전략
  - 리샤딩 수행 시간대/속도 제어

## Deliverables
- 실행 스크립트/절차
- 리샤딩 타임라인 리포트
- CloudWatch 대시보드 스냅샷/JSON
- Runbook(사전 체크/중단 기준/사후 검증)

## Acceptance Criteria
- 리샤딩 전/중/후 지표가 동일 포맷으로 수집되고 자동 비교 가능
- 목표 SLO 만족 가능한 클라이언트 설정 템플릿 1개 이상 도출
