# B-0363 — E-045 Multi-AZ Auto-Failover Drill (ElastiCache test-failover)

## Goal
ElastiCache 표준 방식인 `test-failover` API로 장애 전환을 훈련하고 애플리케이션 영향도를 계량한다.

## Scope
- 실험 문서: `experiments/E-045-elasticache-test-failover-drill.md`
- baseline 부하 측정 후 failover drill 수행
- endpoint 타입별(Primary/Reader/Configuration) 영향 비교
- 관측 항목:
  - reconnect 시간/횟수, write error/timeout, p99 latency
  - (가능 시) read-your-writes 위반 여부
  - CloudWatch 공통 지표 세트

## Deliverables
- AWS CLI 기반 재현 스크립트(`test-failover` 포함)
- failover 타임라인 리포트(초 단위)
- 알람→조치→복구 확인 runbook

## Acceptance Criteria
- failover 동안 영향 구간이 계량 가능하고 재시도 폭주 방지 설정이 검증됨
- 운영 표준 템플릿(재시도/백오프/타임아웃) 확정
