# B-0364 — E-046 Connection Storm / Reconnect Thundering Herd

## Goal
Failover/네트워크 블립 시 동시 재연결 폭주를 재현하고 표준 방어책(backoff+jitter, warmup, CB, pool 제한)을 확정한다.

## Scope
- 실험 문서: `experiments/E-046-elasticache-connection-storm-and-backoff.md`
- 앱 replica 20~200 동시 기동/재연결 시나리오
- 장애 주입:
  - E-045 failover 연계
  - (가능 시) 네트워크 지연/드랍
- 관측:
  - CloudWatch: CurrConnections, EngineCPUUtilization, NetworkBytesIn/Out, FreeableMemory, Evictions
  - App: connect latency, retry/error rate

## Deliverables
- reconnect 표준 파라미터(runbook)
- degrade 전략(필수/선택 기능 분리)
- 자동 검증 스크립트 1개 이상

## Acceptance Criteria
- 방어책 ON/OFF 비교에서 연결 폭주 완화가 정량적으로 확인됨
