# B-0368 — E-050 ElastiCache Serverless vs Node-based Semantics Gap

## Goal
Serverless vs node-based 의미 차이(FLUSH 범위, keyspace notify 제약, 트랜잭션 제약)를 실험해 전환 체크리스트를 확정한다.

## Scope
- 실험 문서: `experiments/E-050-elasticache-serverless-vs-nodebased-semantics.md`
- 동일 테스트 스위트 양 환경 실행
- 항목별 차이 수집:
  - FLUSHDB/FLUSHALL 영향 범위
  - keyspace notification 지원 여부
  - 트랜잭션/운영 제약

## Deliverables
- Serverless 전환 체크리스트
- 금지/주의 동작 목록
- 비교 리포트(동일 포맷)

## Acceptance Criteria
- 양 환경 의미 차이가 재현 가능하게 문서/테스트로 정리됨
- 전환 시 위험 포인트와 우회 전략이 runbook에 반영됨
