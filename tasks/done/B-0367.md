# B-0367 — E-049 ElastiCache Restricted Commands Compatibility Gate

## Goal
ElastiCache 제한 커맨드(CONFIG/BGSAVE/BGREWRITEAOF/MIGRATE/SHUTDOWN 등) 사용을 CI에서 사전 차단한다.

## Scope
- 실험 문서: `experiments/E-049-elasticache-restricted-commands-compat-gate.md`
- denylist/allowlist 정의
- 코드 스캔 룰(문자열/호출 경로) 구현
- CI gate 추가 + 예외 승인 절차 문서화

## Deliverables
- denylist/allowlist 문서
- CI 검사 스크립트/테스트
- 운영 표준: "ElastiCache에서 금지 동작"

## Acceptance Criteria
- 금지 커맨드 호출이 CI에서 실패로 탐지됨
- 최소 1개 의도적 위반 케이스를 테스트로 검증
