# B-0365 — E-047 Slow Log + Log Delivery 기반 p99 원인 분석 표준

## Goal
p99 상승을 느린 명령/빅키/블로킹 패턴으로 재현하고 Slow log + CloudWatch Logs 기반 분석 루틴을 표준화한다.

## Scope
- 실험 문서: `experiments/E-047-elasticache-slowlog-log-delivery-p99.md`
- 파라미터 그룹으로 slowlog 설정, log delivery 활성화
- 느린 요청 주입 후 p99 spike와 원인 커맨드 상관 분석
- 대응책 재측정: big key 분할, 금지 패턴, 워크로드 분리

## Deliverables
- p99 진단 runbook(CW 알람→로그 분석→원인 분류→조치)
- 금지 패턴/가드레일 목록
- 자동 리포트 스크립트

## Acceptance Criteria
- p99 spike 구간의 원인 커맨드 패턴이 로그로 식별 가능
- 대응책 적용 후 개선폭이 수치로 입증됨
