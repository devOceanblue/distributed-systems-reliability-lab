# B-0332 — Chaos/Failure Toolkit 스크립트화 (Broker Kill/Netem/Service Kill)

## Goal
실험에서 “타이밍 운”을 제거하고, 장애를 버튼처럼 재현한다.

## Scope
- `scripts/chaos/` 추가
- 기능:
  1) broker stop/start
  2) 특정 컨테이너 네트워크 지연/패킷드랍(netem) 주입/해제
  3) 서비스 프로세스 kill/restart
  4) redis flush / ttl 조작(스탬피드 실험 보조)
  5) mysql 부하 유도(선택: sysbench 또는 쿼리 폭탄)

## Deliverables
- `scripts/chaos/broker-stop.sh`, `broker-start.sh`
- `scripts/chaos/netem-delay.sh`, `netem-clear.sh`
- `scripts/chaos/kill-service.sh`, `restart-service.sh`
- `Runbook.md`에 사용 예시 추가

## Acceptance Criteria
- E-009에서 브로커 다운 시나리오가 스크립트로 완전 자동화됨
- E-008에서 TTL 만료 타이밍을 스크립트로 재현 가능
